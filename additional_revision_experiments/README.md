# Additional Experiments for the Revised Manuscript

This folder contains the additional experiment code and summarized outputs prepared during the first-round revision of the vegetation segmentation manuscript.

## Contents

- `9_DINOv2/`: DINOv2 four-dataset dense segmentation experiments and smoke-test outputs.
- `10_DINOv3/`: DINOv3 four-dataset dense segmentation experiments.
- `11_QwenVL/`: QwenVL vegetation prompt-evaluation raw outputs.
- `12_LocatingAnything/`: LocateAnything vegetation grounding raw outputs.
- `scripts/`: scripts for dataset preparation, DINO training, prompt-model evaluation, efficiency benchmarking, and perturbation robustness evaluation.
- `results/`: summarized CSV, JSON, and Markdown outputs used for the revised manuscript tables and response materials.
- `*.ipynb`: supplementary training and first-revision experiment notebooks.

## Notes

Large pretrained foundation-model checkpoints and full remote-sensing datasets are not included. The included `.pth` files are lightweight trained segmentation heads generated by the supplementary DINO experiments.

Python cache files and notebook checkpoint files were removed from this GitHub upload copy.
